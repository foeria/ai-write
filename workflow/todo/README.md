## 核心设计原则

### SQLite 唯一数据存储
桌面端所有本地数据操作均使用 SQLite 数据库，不涉及文件操作：
- 小说内容、章节 → SQLite
- 角色信息 → SQLite
- 知识库/词条库 → SQLite
- 用户配置 → SQLite
- AI 对话历史 → SQLite
